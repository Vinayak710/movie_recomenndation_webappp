Title of project : Movie_recomendation_system 

     In this project i have developed an webb app using streamlit library 
     The backind of the app used mashine learning
     In the model the data set used is TMDB top 5000 movies link(https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
     Preprocessing have been done on the data to make it in sutable format 
     Every movie is converted in to Vectore and recomendations are based on nearest vector
